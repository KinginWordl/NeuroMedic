# 🩺 NeuroMedic - Demo Consultorio Médico

Aplicación de escritorio para la gestión de pacientes y generación de recetas médicas, diseñada para consultorios particulares.

## Características
- 🔐 **Login de usuario** (acceso seguro con credenciales)
- 📋 **Gestión de pacientes** (registro, consulta y actualización)
- 📄 **Generación de recetas en PDF** con formato profesional
- 🎨 **Interfaz moderna** con PyQt6
- 🗄️ **Base de datos local** con PostgreSQL

## Tecnologías
- **Python 3** - Lenguaje de programación
- **PyQt6** - Interfaz gráfica de usuario
- **PostgreSQL** - Base de datos relacional
- **WeasyPrint** - Generación de PDFs desde HTML
- **Jinja2** - Motor de plantillas para PDFs
- **psycopg2** - Conector PostgreSQL para Python

## Requisitos del Sistema
- Python 3.8 o superior
- PostgreSQL 15 o superior
- Sistema operativo: Windows, Linux o macOS
